# Unity Simple FPS

- Unity version: 6.1
- Unity Netcode for GameObjects
    - Transport: SteamNetworkingSockets
        - Steamworks.NET

## Features

- Create, Join, Leave Lobby
- Server authoritative
- Client-side prediction
    - Character movement
    - Weapon
